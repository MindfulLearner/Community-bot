# Community-bot

## Perché questo bot?

L'obiettivo principale del progetto è quello di incoraggiare studenti ed ex studenti Boolean ad avvicinarsi al mondo dell'open source, imparare a lavorare in team in maniera strutturata e allenarsi scrivendo codice per qualcosa di concreto: un bot per la community sviluppato dalla community stessa.

## Come contribuire

1. **Fai un fork** di questo repository sul tuo account GitHub
2. **Fai un clone** del fork in locale
3. **Crea un branch** per le tue modifiche (`git checkout -b nome-branch`)  
4. **Apri una Pull Request** per proporre i tuoi cambiamenti

### Pull request

[Video come fare una pull request (in inglese)](https://www.youtube.com/watch?v=rgbCcBNZcdQ)

Per la configurazione del tuo computer e tutto quello che ti serve per far girare il bot in locale, ti rimandiamo alla pagina [CONTRIBUTING](CONTRIBUTING.md)
